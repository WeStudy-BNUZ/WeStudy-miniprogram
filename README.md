# WeStudy-基于Taro(React)+Dva的学习分享社区小程序

## 环境配置
+ **全局安装Taro脚手架：**

  npm install -g @tarojs/cli
+ **安装依赖：**
  
  npm install
## **项目运行：**
  + 开发：
  
    npm run dev:weapp
  + 上线：
  
    NODE_ENV=production taro build --type weapp --watch
